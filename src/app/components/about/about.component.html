About

The Marvel Comics API is a great resource for beginner to advanced developers, as it opens the door of learning how to manage huge amounts of data coming from beautifully constructed API. With NgMarvel we have explored the new Bootstrap 5, Bootswatch theme, Angular Infinite Scroll, Font Awesome, RxJS operators and functions and many more.

This application is about the implementation of Angular application using Marvel Comics API as the data source. The data used in the application belongs to Marvel and used only for learning and exploring the concepts of Angular and about Marvel Comics API. This application is not used as a commercial product.
